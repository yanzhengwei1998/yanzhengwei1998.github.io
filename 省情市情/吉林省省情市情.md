### 2. 区位和邻接

- 吉林省位于东北地区中部。
- 东部与俄罗斯接壤，东南部隔图们江、鸭绿江与朝鲜相望。
- 西接内蒙古自治区，南邻辽宁省，北连黑龙江省。
- 是我国向北开放的重要窗口，也是东北亚区域合作的重要节点。

### 3. 地形和资源

吉林省地形可按“东中西”记忆：

- 东部：长白山区，森林、水资源、矿泉水、人参、生态旅游资源丰富。
- 中部：松辽平原，粮食生产和工业基础集中。
- 西部：草原、湿地、湖泊、盐碱地、新能源资源突出。

重要资源：

| 类型 | 代表内容 | 复习关键词 |
| --- | --- | --- |
| 农业资源 | 黑土地、玉米、水稻、大豆、畜牧业 | 国家重要商品粮基地 |
| 工业资源 | 汽车、石化、轨道客车、医药、装备制造 | 老工业基地转型 |
| 生态资源 | 长白山、松花江、鸭绿江、图们江、湿地 | 生态屏障、绿色发展 |
| 文旅资源 | 长白山、吉林雾凇、查干湖、伪满皇宫、边境民俗 | 冰雪、生态、红色、民俗 |
| 开放资源 | 延边、珲春、图们江、中俄朝交界 | 沿边开放、东北亚合作 |

### 4. 省级近一年政策主线

近一年吉林省各市州政策大体围绕以下主线展开：

- 【重点政策】**实现“十五五”良好开局**：2026年是“十五五”规划开局之年，各地强调高质量发展、项目建设、扩大有效投资。
- 【重点政策】**新时代东北全面振兴**：围绕老工业基地转型、现代化产业体系、营商环境、县域经济发力。
- 【重点政策】**长春现代化都市圈建设**：长春、吉林、四平、辽源等城市加强产业协同、交通互联、文旅联动和公共服务共享。
- 【重点政策】**粮食安全和黑土地保护**：中西部市州突出稳面积、提单产、强科技、保春耕、建高标准农田。
- 【重点政策】**新能源和绿色转型**：白城、松原、辽源等地突出风光电、绿氢、绿色甲醇、零碳园区和绿电消纳。
- 【重点政策】**冰雪文旅和避暑休闲**：长春、吉林、白山、延边、松原等地发展冰雪旅游、民俗旅游、生态康养、边境旅游。

### 1. 基本市情

- 【重点区位】位于吉林省西北部。
- 【重点生态】草原、湿地、湖泊、盐碱地资源突出。
- 【重点保护地】向海、莫莫格等湿地自然保护区知名度高。
- 【重点方向】新能源、现代农业、生态旅游、盐碱地治理。

### 2. 产业和文旅

| 类型 | 内容 |
| --- | --- |
| 代表产业 | 风能、太阳能、绿氢、现代农业、生态畜牧业、湿地生态旅游 |
| 文旅资源 | 向海国家级自然保护区、莫莫格国家级自然保护区、鹤乡、湿地草原风光 |
| 特色资源 | 风光资源、湿地、盐碱地、草原、鹤乡 |

### 3. 近一年政策

- 【重点时间】**2026年**：把新能源、农牧产品、特色旅游作为高质量发展支点。
- 加快从能源输出基地向全产业链绿电消纳基地跨越。
- 推进绿色甲醇、绿氢等项目，打造全省绿色液体燃料供应重镇。
- 围绕绿氢产业生态，同步布局电解槽、碳纤维叶片等装备制造。
- 推进现代化大农业、农牧产品加工、盐碱地综合利用。
- 依托向海、莫莫格、湿地草原和鹤乡资源，发展生态旅游。

## 十、延边朝鲜族自治州

### 1. 基本州情

- 【重点性质】**吉林省唯一的自治州**。
- 【重点首府】**延吉市**。
- 【重点区位】位于吉林省东部，地处中俄朝三国交界区域。
- 【重点民族】朝鲜族聚居地区，民族文化特色鲜明。
- 【重点开放】吉林省向东开放、参与东北亚区域合作的重要前沿。

### 2. 产业和文旅

| 类型 | 内容 |
| --- | --- |
| 代表产业 | 旅游业、食品和农产品加工、医药健康、林下经济、民族文化产业、跨境贸易和现代物流 |
| 文旅资源 | 长白山相关资源、图们江边境风光、延吉民俗文化、珲春防川“一眼望三国” |
| 开放资源 | 珲春、图们江、中俄朝交界、对俄对朝经贸合作 |

### 3. 近一年政策

- 【重点时间】**2025年-2026年**：推进边境民生保障和改善，聚焦教育、医疗、就业、社保、基础设施等。
- 依托G331沿线资源，完善旅游基础设施，培育红色文旅、生态观光、特色驿站等业态。
- 强化重点文旅项目建设和假日旅游安全生产，发展民宿、乡村旅游、旅居康养。
- 推进抽水蓄能等重大项目建设，服务能源安全和绿色发展。
- 发挥珲春、图们江、中俄朝交界区位优势，推进跨境贸易、边境旅游、东北亚合作。

## 十三、申论和面试通用框架

### 1. 谈吉林全面振兴

可用框架：

- 以产业振兴夯实基础：汽车、石化、轨道客车、医药、农产品加工。
- 以科技创新培育新质生产力：光电信息、新材料、新能源、数字经济。
- 以农业现代化保障粮食安全：黑土地保护、高标准农田、农产品精深加工。
- 以生态优势转化发展优势：长白山、湿地、冰雪、康养、文旅融合。
- 以开放合作拓展空间：东北亚合作、沿边开放、图们江开发。
- 以民生改善凝聚振兴动力：就业、教育、医疗、养老、基层治理。

### 2. 谈黑土地保护

可用框架：

- 坚持保护优先，守住耕地红线。
- 推进高标准农田建设。
- 强化科技赋能和数字农业。
- 推进秸秆还田、保护性耕作。
- 完善农田水利和防灾减灾体系。
- 让黑土地持续服务国家粮食安全。

### 3. 谈冰雪经济

可用框架：

- 依托长白山、吉林雾凇、查干湖冬捕等资源。
- 推动冰雪旅游、冰雪运动、冰雪装备、冰雪文化融合发展。
- 提升交通、住宿、服务、数字营销水平。
- 打造四季旅游产品，避免季节性单一。
- 以冰雪经济带动消费和就业。

### 4. 谈沿边开放

可用框架：

- 发挥延边、通化、白山等沿边区位优势。
- 完善口岸、通道、物流、园区功能。
- 深化对俄、对朝及东北亚经贸合作。
- 发展边境旅游、跨境电商、现代物流。
- 在开放中维护边境安全和民族团结。

### 5. 谈生态保护和绿色发展

可用框架：

- 东部保护长白山生态屏障。
- 中部推进工业绿色转型。
- 西部加强湿地、草原、盐碱地治理。
- 发展清洁能源、生态旅游、绿色农业。
- 实现生态价值、经济价值、民生价值统一。
