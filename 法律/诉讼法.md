# 诉讼法

> 分类：法律 / 诉讼法 / 公共基础知识  
> 适用：事业单位、公务员、公基、行政能力常识  
> 整理时间：2026-05-20  
> 说明：本篇先整理三大诉讼基础对比，后续可继续补充管辖、证据、审级、期限和执行。

## 一、三大诉讼比较

| 类型 | 解决问题 | 主管机关 |
| --- | --- | --- |
| 民事诉讼 | 平等主体之间的民事争议 | 人民法院 |
| 行政诉讼 | 公民、法人或其他组织与行政机关之间的行政争议 | 人民法院 |
| 刑事诉讼 | 追究犯罪嫌疑人、被告人刑事责任 | 公安机关、检察机关、人民法院等 |

## 二、举证责任速记

- 民事诉讼：通常“谁主张，谁举证”。
- 行政诉讼：行政机关通常对作出的行政行为负举证责任。
- 刑事诉讼：公诉案件由检察机关承担指控犯罪的证明责任。

## 三、诉讼主体速记

| 诉讼类型 | 常见主体 |
| --- | --- |
| 民事诉讼 | 原告、被告、第三人 |
| 行政诉讼 | 原告通常是行政相对人，被告通常是行政机关 |
| 刑事诉讼 | 公安机关、人民检察院、人民法院、犯罪嫌疑人、被告人、被害人等 |

## 四、高频易混点

| 易混内容 | 正确区分 |
| --- | --- |
| 民事诉讼和行政诉讼 | 民事诉讼解决平等主体争议，行政诉讼解决行政争议 |
| 行政复议和行政诉讼 | 行政复议由行政机关处理，行政诉讼由人民法院处理 |
| 行政诉讼举证责任 | 行政机关通常对作出的行政行为负举证责任 |
| 刑事诉讼证明责任 | 公诉案件由检察机关承担指控犯罪的证明责任 |

## 五、管辖基础

### 1. 民事诉讼管辖

常考规则：

- 一般地域管辖通常遵循“原告就被告”。
- 合同纠纷可关注被告住所地、合同履行地。
- 不动产纠纷通常由不动产所在地人民法院管辖。

### 2. 行政诉讼管辖

常考规则：

- 一般由最初作出行政行为的行政机关所在地人民法院管辖。
- 经复议的案件，也可以由复议机关所在地人民法院管辖。
- 对限制人身自由的行政强制措施不服的，可关注被告所在地或原告所在地管辖。

### 3. 刑事诉讼管辖

常考规则：

- 刑事案件通常由犯罪地人民法院管辖。
- 被告人居住地人民法院审判更适宜的，可以由被告人居住地人民法院管辖。

## 六、证据基础

| 诉讼类型 | 常见证据 |
| --- | --- |
| 民事诉讼 | 当事人陈述、书证、物证、视听资料、电子数据、证人证言、鉴定意见、勘验笔录 |
| 行政诉讼 | 书证、物证、视听资料、电子数据、证人证言、鉴定意见、勘验笔录、现场笔录等 |
| 刑事诉讼 | 物证、书证、证人证言、被害人陈述、犯罪嫌疑人和被告人供述与辩解、鉴定意见、勘验检查辨认侦查实验笔录、视听资料、电子数据 |

易混点：行政诉讼中特别常见“现场笔录”；刑事诉讼中有“犯罪嫌疑人、被告人供述和辩解”。

## 七、审级和程序速记

| 内容 | 速记 |
| --- | --- |
| 两审终审 | 我国诉讼通常实行两审终审制 |
| 民事诉讼 | 起诉、受理、审理、裁判、执行 |
| 行政诉讼 | 起诉、受理、审理、判决；重点审查行政行为合法性 |
| 刑事诉讼 | 立案、侦查、起诉、审判、执行 |

## 八、执行基础

- 民事执行：生效民事判决、裁定、调解书等可依法申请执行。
- 行政执行：行政机关可依法申请人民法院强制执行，法律授权的也可自行强制执行。
- 刑事执行：刑罚执行涉及监狱、看守所、社区矫正机构等。
