## 二、公民基本权利

常考权利：

- 平等权。
- 选举权和被选举权。
- 政治自由。
- 宗教信仰自由。
- 人身自由。
- 人格尊严。
- 住宅不受侵犯。
- 通信自由和通信秘密。
- 监督权。
- 劳动权。
- 休息权。
- 受教育权。

## 三、公民基本义务

常考义务：

- 维护国家统一和全国各民族团结。
- 遵守宪法和法律。
- 维护祖国安全、荣誉和利益。
- 保卫祖国、依法服兵役和参加民兵组织。
- 依法纳税。

## 四、国家机构

### 1. 全国人民代表大会

全国人民代表大会是最高国家权力机关。

常考职权：

- 修改宪法。
- 监督宪法实施。
- 制定和修改刑事、民事、国家机构的和其他的基本法律。
- 选举、决定和罢免有关国家机关组成人员。
- 审查和批准国民经济和社会发展计划、国家预算。
- 决定战争和和平问题。

### 2. 全国人大常委会

全国人大常委会是全国人民代表大会的常设机关。

常考职权：

- 解释宪法，监督宪法实施。
- 制定和修改除应由全国人大制定的法律以外的其他法律。
- 解释法律。
- 监督国务院、中央军事委员会、国家监察委员会、最高人民法院、最高人民检察院的工作。
- 决定特赦。
- 决定全国总动员或者局部动员。

### 3. 国务院

国务院即中央人民政府，是最高国家权力机关的执行机关，是最高国家行政机关。

常考职权：

- 根据宪法和法律，规定行政措施，制定行政法规，发布决定和命令。
- 领导和管理经济、教育、科学、文化、卫生、民政、公安、司法行政等工作。
- 编制和执行国民经济和社会发展计划、国家预算。
- 领导和管理国防建设事业。

### 4. 监察委员会、法院和检察院

| 机关 | 性质 | 速记 |
| --- | --- | --- |

## 六、选举制度

常考点：

- 我国选举制度坚持普遍性、平等性、直接选举和间接选举并用、无记名投票等原则。
- 县级及以下人大代表通常由选民直接选举产生。
- 县级以上地方各级人大代表通常由下一级人大选举产生。
- 依法被剥夺政治权利的人没有选举权和被选举权。

## 七、特别行政区制度

重点理解：

- 特别行政区制度是“一国两制”方针的制度安排。
- 特别行政区享有高度自治权，但不享有国家主权。
- 中央对特别行政区拥有全面管治权。

## 八、高频易混点

| 易混内容 | 正确区分 |
| --- | --- |
| 宪法和普通法律 | 宪法是根本法，具有最高法律效力 |
| 国体和政体 | 国体是人民民主专政，政体是人民代表大会制度 |
| 权利和义务 | 权利强调依法享有，义务强调依法履行 |
| 全国人大和全国人大常委会 | 全国人大是最高国家权力机关，全国人大常委会是其常设机关 |
| 法院和检察院 | 法院是审判机关，检察院是法律监督机关 |
| 居委会、村委会 | 基层群众性自治组织，不是基层政权机关 |
| 特别行政区高度自治权 | 高度自治不等于完全自治，不享有国家主权 |
