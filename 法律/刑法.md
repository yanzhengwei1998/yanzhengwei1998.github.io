# 刑法

> 分类：法律 / 刑法 / 公共基础知识  
> 适用：事业单位、公务员、公基、行政能力常识  
> 整理时间：2026-05-20  
> 说明：本篇先整理刑法基础，后续可继续补充犯罪形态、共同犯罪、重点罪名等案例考点。

## 一、犯罪构成

犯罪构成四要件：

1. 犯罪主体。
2. 犯罪主观方面。
3. 犯罪客体。
4. 犯罪客观方面。

速记：主体、主观、客体、客观。

## 二、正当防卫

正当防卫是为了使国家、公共利益、本人或他人的人身、财产和其他权利免受正在进行的不法侵害，而采取的制止不法侵害的行为。

关键词：

- 正在进行的不法侵害。
- 防卫对象是不法侵害人。
- 没有明显超过必要限度造成重大损害。

## 三、紧急避险

紧急避险是为了使国家、公共利益、本人或他人的人身、财产和其他权利免受正在发生的危险，不得已采取损害另一较小合法权益的行为。

易混点：

- 正当防卫针对“不法侵害人”。
- 紧急避险通常损害“第三方较小合法权益”。

## 四、刑罚种类

| 类型 | 内容 |
| --- | --- |
| 主刑 | 管制、拘役、有期徒刑、无期徒刑、死刑 |
| 附加刑 | 罚金、剥夺政治权利、没收财产、驱逐出境 |

## 五、犯罪主观方面

| 类型 | 备考理解 |
| --- | --- |
| 直接故意 | 明知会发生危害结果，并希望结果发生 |
| 间接故意 | 明知会发生危害结果，并放任结果发生 |
| 疏忽大意的过失 | 应当预见而没有预见 |
| 过于自信的过失 | 已经预见但轻信能够避免 |

## 六、犯罪形态

| 形态 | 关键词 |
| --- | --- |
| 犯罪预备 | 为犯罪准备工具、制造条件 |
| 犯罪未遂 | 已经着手实行犯罪，因意志以外原因未得逞 |
| 犯罪中止 | 自动放弃犯罪或自动有效防止犯罪结果发生 |
| 犯罪既遂 | 犯罪构成要件齐备 |

易混点：未遂是“想成而没成”，中止是“能继续而主动停”。

## 七、共同犯罪

共同犯罪是二人以上共同故意犯罪。

常见角色：

- 主犯：组织、领导犯罪集团进行犯罪活动，或在共同犯罪中起主要作用。
- 从犯：在共同犯罪中起次要或辅助作用。
- 胁从犯：被胁迫参加犯罪。
- 教唆犯：教唆他人犯罪。

## 八、量刑情节

| 情节 | 备考理解 |
| --- | --- |
| 自首 | 犯罪以后自动投案，如实供述自己的罪行 |
| 立功 | 揭发他人犯罪行为、提供重要线索等经查证属实 |
| 累犯 | 受过一定刑罚处罚，在法定期限内又犯罪 |
| 数罪并罚 | 一人犯数罪时依法合并处罚 |

## 九、常见重点罪名

| 罪名 | 速记 |
| --- | --- |
| 盗窃罪 | 秘密窃取公私财物 |
| 诈骗罪 | 虚构事实、隐瞒真相，使他人陷入错误认识并处分财产 |
| 抢劫罪 | 以暴力、胁迫或其他方法当场劫取财物 |
| 抢夺罪 | 乘人不备，公然夺取财物 |
| 交通肇事罪 | 违反交通运输管理法规，发生重大事故 |
| 危险驾驶罪 | 追逐竞驶、醉酒驾驶等危险驾驶行为 |
| 贪污罪 | 国家工作人员利用职务便利非法占有公共财物 |
| 受贿罪 | 国家工作人员利用职务便利索取或非法收受财物，为他人谋取利益 |

## 十、高频易混点

| 易混内容 | 正确区分 |
| --- | --- |
| 正当防卫和紧急避险 | 正当防卫针对不法侵害人，紧急避险通常损害较小合法权益 |
| 行政拘留和拘役 | 行政拘留是行政处罚，拘役是刑罚主刑 |
| 罚款和罚金 | 罚款多为行政处罚，罚金是刑罚附加刑 |
| 主刑和附加刑 | 主刑只能独立适用，附加刑可以附加适用也可以独立适用 |
| 盗窃罪和抢夺罪 | 盗窃通常秘密窃取，抢夺通常公然夺取 |
| 抢夺罪和抢劫罪 | 抢劫使用暴力、胁迫或其他方法压制反抗 |
| 犯罪未遂和犯罪中止 | 未遂因意志以外原因未得逞，中止是自动放弃或自动防止结果 |
