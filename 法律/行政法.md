# 行政法

> 分类：法律 / 行政法 / 公共基础知识  
> 适用：事业单位、公务员、公基、行政执法类岗位  
> 整理时间：2026-05-20  
> 说明：本篇先整理行政法基础，后续可继续补充行政许可、行政强制、国家赔偿、政府信息公开等专题。

## 一、行政法核心

行政法主要规范行政权的设定、运行、监督和救济。

备考关键词：

- 行政主体。
- 行政行为。
- 行政相对人。
- 依法行政。
- 程序正当。
- 权责统一。
- 行政救济。

## 二、常见行政行为

| 行为 | 含义 |
| --- | --- |
| 行政许可 | 行政机关根据申请，经审查准予从事特定活动 |
| 行政处罚 | 对违反行政管理秩序的行为依法给予制裁 |
| 行政强制 | 对人身、财产或行为采取强制措施或强制执行 |
| 行政征收 | 国家依法无偿取得相对人财产所有权 |
| 行政征用 | 紧急需要时依法临时使用相对人财产 |
| 行政确认 | 对法律事实、法律关系进行确认 |
| 行政裁决 | 行政机关裁处特定民事争议 |

## 三、行政处罚种类

常见行政处罚包括：

- 警告、通报批评。
- 罚款、没收违法所得、没收非法财物。
- 暂扣许可证件、降低资质等级、吊销许可证件。
- 限制开展生产经营活动、责令停产停业、责令关闭、限制从业。
- 行政拘留。
- 法律、行政法规规定的其他行政处罚。

## 四、行政复议和行政诉讼

| 项目 | 行政复议 | 行政诉讼 |
| --- | --- | --- |
| 性质 | 行政机关内部层级监督和救济 | 人民法院司法审查 |
| 审查对象 | 具体行政行为等行政争议 | 行政行为合法性 |
| 结果形式 | 行政复议决定 | 行政判决、裁定 |

易混点：行政复议由行政机关处理，行政诉讼由人民法院处理。

## 五、行政许可

### 1. 含义

行政许可是行政机关根据公民、法人或者其他组织的申请，经依法审查，准予其从事特定活动的行为。

### 2. 常考原则

- 法定原则。
- 公开、公平、公正、非歧视原则。
- 便民原则。
- 救济原则。
- 信赖保护原则。
- 监督原则。

### 3. 典型例子

- 市场主体登记。
- 食品经营许可。
- 建设工程规划许可。
- 驾驶证许可。

## 六、行政强制

| 类型 | 含义 | 例子 |
| --- | --- | --- |
| 行政强制措施 | 为制止违法、防止证据损毁、避免危害发生等，对人身或财物实施暂时性控制 | 查封、扣押、冻结 |
| 行政强制执行 | 行政相对人不履行行政决定时，依法强制履行义务 | 加处罚款、划拨存款、代履行 |

易混点：行政强制措施重在“暂时控制”，行政强制执行重在“实现义务”。

## 七、国家赔偿

### 1. 行政赔偿

行政机关及其工作人员违法行使行政职权，侵犯公民、法人或其他组织合法权益并造成损害的，依法承担行政赔偿责任。

常见情形：

- 违法拘留或者违法采取限制人身自由的行政强制措施。
- 非法拘禁或者以其他方法非法剥夺公民人身自由。
- 违法罚款、吊销许可证和执照、责令停产停业、没收财物等。
- 违法征收、征用财产。

### 2. 刑事赔偿

司法机关及其工作人员在刑事诉讼中违法行使职权造成损害，符合条件的依法承担刑事赔偿责任。

## 八、政府信息公开

常考点：

- 行政机关应当主动公开涉及公众利益调整、需要公众广泛知晓或者需要公众参与决策的政府信息。
- 公民、法人或其他组织可以依法申请获取相关政府信息。
- 涉及国家秘密、商业秘密、个人隐私等信息，依法受到限制。

## 九、行政诉讼重点

### 1. 受案范围速记

常见可诉行政行为：

- 对行政处罚不服。
- 对行政强制措施和行政强制执行不服。
- 申请行政许可被拒绝或行政机关不予答复。
- 行政机关不依法履行保护人身权、财产权等法定职责。
- 认为行政机关侵犯经营自主权、农村土地承包经营权等。
- 认为行政机关违法要求履行义务。

### 2. 举证责任

行政诉讼中，被告行政机关通常对作出的行政行为负举证责任。

## 十、高频易混点

| 易混内容 | 正确区分 |
| --- | --- |
| 行政处罚和刑罚 | 行政处罚由行政机关依法实施，刑罚由人民法院依法判处 |
| 行政复议和行政诉讼 | 行政复议找行政机关，行政诉讼找人民法院 |
| 行政征收和行政征用 | 征收改变所有权，征用通常是临时使用并补偿 |
| 行政许可和行政确认 | 许可是准予从事特定活动，确认是确认事实或关系 |
| 行政强制措施和行政强制执行 | 措施是暂时控制，执行是实现已确定义务 |
| 行政赔偿和行政补偿 | 赔偿通常源于违法侵权，补偿通常源于合法行为造成特别损失 |
