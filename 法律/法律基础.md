## 二、法的基本概念

### 1. 法的含义

法是由国家制定或认可，并由国家强制力保证实施的行为规范体系。

### 2. 法的基本特征

| 特征 | 解释 |
| --- | --- |
| 国家意志性 | 法体现由国家确认的意志 |
| 规范性 | 法规定人们可以、应当或不得怎样行为 |
| 普遍约束力 | 在法定范围内对相关主体普遍适用 |
| 国家强制性 | 由国家强制力保障实施 |
| 程序性 | 法的制定、适用、执行通常遵循法定程序 |

### 3. 法的作用

| 作用 | 关键词 |
| --- | --- |
| 指引作用 | 指引本人行为 |
| 评价作用 | 评价他人行为是否合法 |
| 教育作用 | 通过法律实施教育公众 |
| 预测作用 | 预测行为后果 |
| 强制作用 | 制裁违法犯罪行为 |

易混点：指引作用主要针对本人行为；评价作用主要针对他人行为。

## 三、法律关系

### 1. 法律关系三要素

| 要素 | 内容 |
| --- | --- |
| 主体 | 参加法律关系、享有权利和承担义务的人或组织 |
| 内容 | 法律权利和法律义务 |
| 客体 | 权利义务指向的对象，如物、行为、智力成果、人身利益等 |

### 2. 法律事实

法律事实是能够引起法律关系产生、变更或消灭的事实。

| 类型 | 含义 | 例子 |
| --- | --- | --- |
| 法律事件 | 不以人的意志为转移 | 出生、死亡、自然灾害 |
| 法律行为 | 以人的意志为转移 | 签合同、结婚、侵权 |

## 四、法律责任

### 1. 法律责任类型

| 类型 | 常见情形 |
| --- | --- |
| 民事责任 | 违约责任、侵权责任 |
| 行政责任 | 行政处罚、行政处分 |
| 刑事责任 | 犯罪后承担刑罚 |
| 违宪责任 | 国家机关及其工作人员违反宪法 |

### 2. 归责原则速记

- 民事责任强调平等主体之间的权利救济。
- 行政责任强调行政管理秩序和公权力规制。
- 刑事责任强调犯罪行为和刑罚制裁。

## 五、依法治国基础

### 1. 全面依法治国

全面依法治国是国家治理的一场深刻革命。

【重点目标】**建设中国特色社会主义法治体系、建设社会主义法治国家**。

### 2. 法治体系

中国特色社会主义法治体系主要包括：

1. 完备的法律规范体系。
2. 高效的法治实施体系。
3. 严密的法治监督体系。
4. 有力的法治保障体系。
5. 完善的党内法规体系。

### 3. 法治原则

| 原则 | 备考理解 |
| --- | --- |
| 法律面前人人平等 | 平等享有权利、平等履行义务、平等受到保护和追究 |
| 权力法定 | 国家机关权力来源于法律授权 |
| 程序正当 | 行政、司法等活动应遵守法定程序 |
| 权责统一 | 有权必有责，用权受监督，违法要追究 |
| 尊重和保障人权 | 贯穿立法、执法、司法、守法全过程 |

## 六、高频易混点

| 易混内容 | 正确区分 |
| --- | --- |
| 法律事件和法律行为 | 事件不以人的意志为转移，行为以人的意志为转移 |
| 民事责任和行政责任 | 民事责任多发生于平等主体之间，行政责任多发生于行政管理关系 |
| 行政处罚和刑罚 | 行政处罚由行政机关依法实施，刑罚由人民法院依法判处 |
