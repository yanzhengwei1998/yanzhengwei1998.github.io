# 民法

> 分类：法律 / 民法 / 公共基础知识  
> 适用：事业单位、公务员、公基、行政能力常识  
> 整理时间：2026-05-20  
> 说明：本篇先整理民法基础，后续可继续补充物权、合同、婚姻家庭、继承、侵权责任等案例考点。

## 一、民法调整对象

民法调整平等主体的自然人、法人和非法人组织之间的人身关系和财产关系。

关键词：

- 平等主体。
- 人身关系。
- 财产关系。
- 意思自治。
- 民事权利保护。

## 二、民事主体

| 主体 | 重点 |
| --- | --- |
| 自然人 | 民事权利能力、民事行为能力、监护 |
| 法人 | 营利法人、非营利法人、特别法人 |
| 非法人组织 | 不具有法人资格，但能依法以自己名义从事民事活动 |

## 三、民事行为能力

| 类型 | 备考理解 |
| --- | --- |
| 完全民事行为能力人 | 通常可以独立实施民事法律行为 |
| 限制民事行为能力人 | 实施与年龄、智力、精神健康状况相适应的行为有效 |
| 无民事行为能力人 | 由法定代理人代理实施民事法律行为 |

## 四、民法典七编

【重点文件】**《中华人民共和国民法典》**包括：

1. 总则。
2. 物权。
3. 合同。
4. 人格权。
5. 婚姻家庭。
6. 继承。
7. 侵权责任。

速记：总物合人婚继侵。

## 五、民事法律行为

### 1. 有效要件

民事法律行为有效通常需要：

- 行为人具有相应的民事行为能力。
- 意思表示真实。
- 不违反法律、行政法规的强制性规定，不违背公序良俗。

### 2. 效力类型

| 类型 | 备考理解 |
| --- | --- |
| 有效 | 依法产生当事人预期的民事法律后果 |
| 无效 | 自始没有法律约束力 |
| 可撤销 | 因重大误解、欺诈、胁迫、显失公平等可请求撤销 |
| 效力待定 | 需要权利人追认后才确定效力 |

### 3. 代理

| 类型 | 说明 |
| --- | --- |
| 委托代理 | 基于被代理人授权 |
| 法定代理 | 基于法律规定，如监护人代理未成年人 |
| 无权代理 | 没有代理权、超越代理权或代理权终止后实施代理行为 |
| 表见代理 | 虽无代理权，但相对人有理由相信其有代理权 |

## 六、物权

| 类型 | 内容 |
| --- | --- |
| 所有权 | 对自己的不动产或动产依法享有占有、使用、收益、处分的权利 |
| 用益物权 | 对他人所有的不动产或动产依法占有、使用、收益 |
| 担保物权 | 为担保债权实现而设立，如抵押权、质权、留置权 |

常考点：

- 不动产物权变动通常以登记为公示方式。
- 动产物权变动通常以交付为公示方式。
- 抵押不转移占有，质押通常转移占有。

## 七、合同

### 1. 合同订立

合同订立通常包括要约和承诺。

| 概念 | 备考理解 |
| --- | --- |
| 要约 | 希望和他人订立合同的意思表示 |
| 承诺 | 受要约人同意要约的意思表示 |
| 要约邀请 | 希望他人向自己发出要约，如拍卖公告、招标公告、商业广告一般情形 |

### 2. 合同履行原则

- 全面履行。
- 诚信履行。
- 协作履行。
- 绿色原则。

### 3. 违约责任

常见承担方式：

- 继续履行。
- 采取补救措施。
- 赔偿损失。
- 支付违约金。
- 定金罚则。

易混点：定金具有担保性质；订金通常不当然适用定金罚则。

## 八、婚姻家庭和继承

### 1. 婚姻家庭

常考点：

- 婚姻自由、一夫一妻、男女平等。
- 夫妻在婚姻家庭中地位平等。
- 家庭成员应当敬老爱幼、互相帮助。
- 离婚方式包括协议离婚和诉讼离婚。

### 2. 继承

| 类型 | 说明 |
| --- | --- |
| 法定继承 | 按法律规定的继承人范围、顺序继承 |
| 遗嘱继承 | 按被继承人生前合法有效遗嘱继承 |
| 遗赠 | 自然人将个人财产赠与国家、集体或法定继承人以外组织、个人 |
| 遗赠扶养协议 | 扶养人承担生养死葬义务并享有受遗赠权利 |

常考顺序：

- 第一顺序：配偶、子女、父母。
- 第二顺序：兄弟姐妹、祖父母、外祖父母。

## 九、侵权责任

### 1. 一般构成

侵权责任常见构成要件：

- 侵权行为。
- 损害后果。
- 因果关系。
- 过错。

### 2. 常见特殊侵权

| 类型 | 备考关键词 |
| --- | --- |
| 产品责任 | 产品存在缺陷造成损害 |
| 机动车交通事故责任 | 交通事故损害赔偿 |
| 医疗损害责任 | 医疗机构及医务人员诊疗活动 |
| 环境污染和生态破坏责任 | 生态环境损害 |
| 饲养动物损害责任 | 动物致人损害 |
| 建筑物和物件损害责任 | 建筑物、构筑物、搁置物、悬挂物等致害 |

## 十、高频易混点

| 易混内容 | 正确区分 |
| --- | --- |
| 民法调整对象 | 平等主体之间的人身关系和财产关系 |
| 法人和非法人组织 | 法人具有法人资格，非法人组织不具有法人资格但可依法以自己名义活动 |
| 民事权利能力和行为能力 | 权利能力强调能否享有权利承担义务，行为能力强调能否独立实施民事法律行为 |
| 要约和要约邀请 | 要约可经承诺成立合同，要约邀请是希望他人向自己发出要约 |
| 抵押和质押 | 抵押通常不转移占有，质押通常转移占有 |
| 定金和订金 | 定金适用定金罚则，订金一般按预付款处理 |
| 法定继承第一顺序 | 配偶、子女、父母 |
