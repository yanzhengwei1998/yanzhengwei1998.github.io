## 一、邓小平理论

### 1. 四个现代化

四个现代化是指工业现代化、农业现代化、国防现代化、科学技术现代化。

| 考点 | 内容 |
| --- | --- |
| 四个现代化 | 工业现代化、农业现代化、国防现代化、科学技术现代化 |
| 常见规范顺序 | 农业现代化、工业现代化、国防现代化、科学技术现代化 |
| 关键环节 | 科学技术现代化 |
| 邓小平重要判断 | 四个现代化，关键是科学技术的现代化 |
| 理论理解 | 科学技术现代化是推动工业、农业、国防现代化的重要支撑 |

易混点：

| 问法 | 答案 |
| --- | --- |
| 四个现代化包括什么 | 工业、农业、国防、科学技术现代化 |
| 实现四个现代化的关键 | 科学技术现代化 |
| 科学技术的地位 | 科学技术是第一生产力 |
