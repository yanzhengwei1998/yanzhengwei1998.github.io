# 马克思主义哲学

> 分类：政治 / 马克思主义哲学 / 公共基础知识  
> 适用：事业单位、公务员、公基、申论、面试  

## 一、唯物辩证法

### 1. 具体问题具体分析

具体问题具体分析，是指在矛盾普遍性原理指导下，具体分析矛盾的特殊性，并找出解决矛盾的正确方法。

| 考点 | 内容 |
| --- | --- |
| 哲学依据 | 矛盾具有特殊性 |
| 规范表述 | 具体问题具体分析是正确认识事物的基础，也是正确解决矛盾的关键 |
| 方法要求 | 不同事物的矛盾用不同方法解决；同一事物在不同阶段的矛盾也要用不同方法解决 |
| 反对倾向 | 反对“一刀切”、主观主义、教条主义、经验主义 |
| 高频关联 | 马克思主义活的灵魂在于具体地分析具体的情况 |

易混点：

| 问法 | 答案 |
| --- | --- |
| 唯物辩证法的根本观点 | 矛盾观点 |
| 矛盾问题的精髓 | 矛盾的普遍性和特殊性、共性和个性的关系 |
| 具体问题具体分析的哲学依据 | 矛盾的特殊性 |
| 正确认识事物的基础 | 具体问题具体分析 |
| 正确解决矛盾的关键 | 具体问题具体分析 |
