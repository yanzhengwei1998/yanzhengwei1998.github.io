## 一、党的性质

### 1. 中国共产党的先锋队性质

中国共产党是中国工人阶级的先锋队，同时是中国人民和中华民族的先锋队。

| 考点 | 内容 |
| --- | --- |
| 阶级基础 | 中国工人阶级 |
| 群众基础 | 中国人民和中华民族 |
| 规范表述 | 中国共产党是中国工人阶级的先锋队，同时是中国人民和中华民族的先锋队 |
| 政治定位 | 中国特色社会主义事业的领导核心 |
| 根本代表 | 代表中国先进生产力的发展要求，代表中国先进文化的前进方向，代表中国最广大人民的根本利益 |

### 2. 党的先进性

中国共产党的历史使命、历史地位、历史作用，始终同党的先进性联系在一起。

| 角度 | 理解 |
| --- | --- |
| 历史使命 | 为中国人民谋幸福、为中华民族谋复兴 |
| 历史地位 | 中国特色社会主义事业的领导核心 |
| 历史作用 | 团结带领中国人民推进革命、建设、改革和新时代伟大实践 |
| 先进性体现 | 理论先进、政治先进、组织先进、作风先进、纪律严明 |
| 考试理解 | 党的先进性是党能够承担历史使命、巩固历史地位、发挥历史作用的重要依据 |
